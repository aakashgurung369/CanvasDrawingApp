### Run Locally

To run **Canva Drawing Website** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/aakashgurung369/CanvasDrawingApp
```

Windows:

```bash
git clone https://github.com/aakashgurung369/CanvasDrawingApp
```
